    <b>Переменные:</b>
Представьте переменные как коробки, в которых вы можете хранить разные вещи. Каждая коробка имеет свое имя, чтобы вы могли легко найти нужную вам вещь.

    <color=#800080ff>Пример:</color>
# Создаем переменные с именами "яблоко" и "конфета" и кладем в них значения
<b>яблоко = 5</b>
<b>конфета = 10</b>

# Можем складывать, вычитать, умножать и делить значения переменных
<b>всего_фруктов</b> = <b>яблоко</b> + <b>конфета</b>
<b>осталось_конфет</b> = <b>конфета - 2</b>

Для именования переменных есть правила - это название переменной в Python должно начинаться с алфавитного символа или со знака подчеркивания и может содержать алфавитно-цифровые символы и знак подчеркивания. И кроме того, название переменной не должно совпадать с названием ключевых слов языка Python. Ключевых слов не так много, их легко запомнить:


False      await      else       import     pass
None       break      except     in         raise
True       class      finally    is         return
and        continue   for        lambda     try
as         def        from       nonlocal   while
assert     del        global     not        with
async      elif       if         or         yield

    <b>Типы данных:</b>
Типы данных определяют, какую информацию можно положить в каждую коробку. Есть разные типы данных для чисел, текста и других вещей.

    <b>Пример:</b>

# Числа - целые числа (int) и числа с плавающей точкой (float)
возраст = 10    # тип данных int (целое число)
рост = 1.35     # тип данных float (число с плавающей точкой)

# Текст - строка (str)
имя = "Анна"    # тип данных str (строка)

# Список - коллекция элементов
фрукты = ["яблоко", "груша", "апельсин"]   # тип данных list (список)

Таким образом, переменные помогают нам хранить и оперировать данными, а типы данных определяют, какой тип информации можно хранить в каждой переменной. Представьте переменные как коробки с ярлыками, а типы данных как разные виды вещей, которые можно положить в каждую коробку.
