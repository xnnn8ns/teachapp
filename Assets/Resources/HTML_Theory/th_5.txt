<b><size=50><color=#8B01AE>Тема:</color><color=#8B01AE>Логические переменные</color></size></b>

В Python имеются следующие логические операторы:<color=#5B0B70><b>and</b>(логическое умножение),<b>or</b>(логическое сложение),<b>not</b>(логическое отрицание).</color>

<color=#5B0B70>and</color>(и):
Этот оператор возвращает значение TRUE только если все переменные имеют значение TRUE. 
Если хотя бы одна из переменных будет FALSE , то и значение будет FALSE.

<b><color=#5B0B70>Пример:</color></b>
age = 23
weight = 50
result = age > 21 and weight == 50
print(result)  # True

В данном случае оператор and сравнивает результаты двух выражений: age > 21 weight == 50. И если оба этих выражений возвращают True,
то оператор and также возвращает True (формально возвращается значение последнего операнда).

<color=#5B0B70>or</color>(или):

Этот оператор возвращает значение TRUE если хотя бы одна переменная имеют значение TRUE. 
Если все из переменных будет FALSE , то только тогда значение будет FALSE.

<b><color=#5B0B70>Пример:</color></b>
age = 25
Married = False
res = age > 21 or Married
print(res)  # True, так как выражение age > 21 равно True

<color=#5B0B70>not</color>(не)

Возвращает True, если выражение равно False и наоборт, возвращает False, если значение True

<b><color=#5B0B70>Пример:</color></b>
age = 25
Married = False
print(not age > 21)  # False
print(not Married)  # True
print(not 5)  # False
print(not 0)  # True


0 эквивалентент False, 1 эквивалентна True
