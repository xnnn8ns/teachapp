<color=#ffff00ff><b><size=55>Типы данных</size></b></color> нужны для того, чтоб определить, какую именно информацию можно хранить в контейнере.
Для чисел, текста и других вещей используются разные типы данных.

Для <color=#add8e6ff><b><size=55>целых чисел</size></b></color> используется тип данных <color=#ffff00ff><b><size=55>int</size></b></color> (165). 
Для <color=#add8e6ff><b><size=55>чисел с плавающей точкой</size></b></color> тип данных <color=#ffff00ff><b><size=55>float</size></b></color> (187.756).
Для <color=#add8e6ff><b><size=55>текста</size></b></color> тип данных <color=#ffff00ff><b><size=55>str</size></b></color> ("Молоко").
Для <color=#add8e6ff><b><size=55>коллекции элементов</size></b></color> тип данных <color=#ffff00ff><b><size=55>list</size></b></color> (["яблоко", "груша", "апельсин"] )

Если представить, что переменные это закрытые коробки с вещами, то тип данных это как стикеры на коробках, которые говорят, что там находится.  

<color=#ffff00ff><b><size=55>Пример:</size></b></color>
<color=#add8e6ff><b>int x=14</b></color>
<color=#add8e6ff><b>str a</b></color>
