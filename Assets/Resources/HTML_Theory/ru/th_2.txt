<color=#5B0B70>Типы данных</color> нужны для того, чтоб определить, какую именно информацию можно хранить в контейнере.
Для чисел, текста и других вещей используются разные типы данных.

Для <b>целых чисел</b> используется тип данных <b>int</b> (165). 
Для <b>чисел с плавающей точкой</b> тип данных <b>float</b> (187.756).
Для <b>текста</b> тип данных <b>str</b> ("Молоко").
Для <b>коллекции элементов</b> тип данных <b>list</b> (["яблоко", "груша", "апельсин"] )

Если представить, что переменные это закрытые коробки с вещами, то тип данных это как стикеры на коробках, которые говорят, что там находится.  

<b><color=#5B0B70>Пример:</color></b>
int x=14
