<b><color=#ffff00ff><size=55>Циклы</size></color></b> позволяют выполнять некоторое действие в зависимости от соблюдения некоторого <b>условия</b>. В языке Python есть следующие <b><color=#add8e6ff>типы циклов:</color></b>
<b><color=#ffff00ff><size=55>while, for</size></color></b>

После ключевого слова <b>while</b> указывается условие, и пока это выражение возвращает значение <b>True</b>, будет выполняться блок инструкций, который идет далее.

<b><color=#ffff00ff><size=55>Конструкция:</size></color></b>
<b>while условное_выражение:
   инструкции</b>

<b><color=#ffff00ff><size=55>Пример:</size></color></b>
<color=#add8e6ff><b>num = 0
while num < 3:
    print("num = {num}")
    num += 1
print("программа завершена")</b></color>

В данном случае цикл <b>while</b> будет выполняться, пока переменная <b>num</b> меньше 3 и выполнится <b>3 раза</b>. 
<color=#add8e6ff><b>Необходимо помнить</b></color> о 4 отступах слева у вычислений внутри цикла.
