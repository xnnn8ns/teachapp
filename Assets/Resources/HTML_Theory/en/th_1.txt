<color=#FFFFFF>Variables</color> are containers in which various data can be stored. 

Variable names can only consist of <b>latin letters, numbers, and underscores.</b> 
The name of the variable <b>cannot</b> start with a digit. 

The language is also <b>case sensitive</b>,  the variables  <b>Tom</b> and <b>tom</b> are different variables.

And besides, the name of the variable should not match the keywords of the language..

<b>Keywords:</b>
false, true, none, else, if, import,pass, class,return,def,and,or,yield

<b><color=#FFFFFF>Example:</color></b>
Correctly named variables look like this:
_Pole, N1ame, mas
