<b><size=50><color=#8B01AE>Тема:</color><color=#8B01AE>Цикл while</color></size></b>

Циклы позволяют выполнять некоторое действие в зависимости от соблюдения некоторого условия. В языке Python есть следующие типы циклов:
<b><color=#8B01AE>while,for</color></b>

После ключевого слова while указывается условие, и пока это выражение возвращает значение True, будет выполняться блок инструкций, который идет далее.

<b><color=#5B0B70>Конструкция:</color></b>
while условное_выражение:
   инструкции

<b><color=#5B0B70>Пример:</color></b>
num = 0
while number < 3:
    print("num = {num}")
    num += 1
print("программа завершена")

В данном случае цикл while будет выполняться, пока переменная number меньше 3 и выполнится 3 раза. 
Необходимо помнить о 4 отступах слева у вычислений внутри цикла.
