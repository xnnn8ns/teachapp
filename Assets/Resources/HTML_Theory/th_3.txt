<b><size=50><color=#8B01AE>Тема:</color>   <color=#8B01AE>Арифметические операторы</color></size></b>

К <color=#5B0B70>арифметическим операторам</color> относятся: <b>сложение (+), вычитание (-), умножение (*), деление (/) </b>.

<b><color=#5B0B70>Пример:</color></b>
сложение= 7+3
значение переменной результат будет равно 10

вычитание= 8-6
значение переменной вычитание будет равно 2

умножение = 4*2
значение переменной умножение  будет равно 8

деление = 9/3
значение переменной деление  будет равно 3

Для деления нужно помнить, что тип данных переменной, в которую записывается результат деления, должен быть 
<b>тип данных <color=#5B0B70>float</color></b> (число с плавающей точкой).
