<color=#FFFFFF>Arithmetic operators</color> include: <b>addition (+), subtraction (-), multiplication (*), division (/)</b>.

<b><color=#FFFFFF>Example:</color></b>
addition= 7+3
the value of the result variable will be 10

subtraction= 8-6
the value of the variable subtraction will be 2

multiplication = 4*2
the value of the variable multiplication will be 8

division = 9/3
the value of the variable division will be 3

For division, it must be remembered that the data type of the variable into which the result of division is written must be
<b>data type <color=#FFFFFF>float</color></b> (floating point number).
